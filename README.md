# site-yuanluo


A personal site showcasing the structure and styling found in [templates/index.html](templates/index.html). It runs on Flask via [`app`](main.py) and has sections for About, Education, Experience, Projects, and Skills, featuring parallax background images, card-hover effects, and more.

## Features
- Hero section with a floating avatar and animated text.
- Dynamic dark/light mode toggle built with Tailwind CSS utilities.
- Responsive mobile navigation for smaller screens.
- Animated skill bars and timeline-style education/experience sections.

## How to Run
1. Install dependencies from [pyproject.toml](pyproject.toml).  
2. Run the site using:
   ```bash
   python main.py
